# Роли и права

Роли предназначены для модулей автодилера (сделки, склад, клиенты, отчёты, финансы). Auth-сервис хранит только роль пользователя; проверка прав выполняется в соответствующих сервисах по JWT (user_id, email, role).

## Роли

| Роль         | Код          | Описание |
|--------------|--------------|----------|
| **Администратор** | `admin`     | Полный доступ: пользователи, роли, настройки, все модули. |
| **Менеджер**      | `manager`   | Управление сделками, складом, отчёты. Назначение ролей sales, accountant, viewer (не admin). |
| **Продавец**      | `sales`     | Свои сделки, просмотр авто и клиентов, создание заявок. |
| **Бухгалтер**     | `accountant`| Финансы: отчёты, платежи, договоры. Просмотр сделок. |
| **Наблюдатель**   | `viewer`    | Только просмотр: справочники, отчёты, без редактирования. |

## Права по ролям

| Право | admin | manager | sales | accountant | viewer |
|-------|-------|---------|-------|------------|--------|
| `users:read` | ✓ | ✓ | — | — | — |
| `users:write` | ✓ | ✓* | — | — | — |
| `users:assign_roles` | ✓ | ✓* | — | — | — |
| `deals:read` | ✓ | ✓ | ✓ | ✓ | ✓ |
| `deals:write` | ✓ | ✓ | ✓ (свои) | — | — |
| `inventory:read` | ✓ | ✓ | ✓ | ✓ | ✓ |
| `inventory:write` | ✓ | ✓ | — | — | — |
| `customers:read` | ✓ | ✓ | ✓ | ✓ | ✓ |
| `customers:write` | ✓ | ✓ | ✓ | — | — |
| `reports:read` | ✓ | ✓ | ✓ | ✓ | ✓ |
| `finance:read` | ✓ | ✓ | — | ✓ | — |
| `finance:write` | ✓ | — | — | ✓ | — |
| `settings:read` | ✓ | ✓ | — | — | — |
| `settings:write` | ✓ | — | — | — | — |

\* Менеджер не может создавать/редактировать админов и не может назначать роль `admin`.

При регистрации новому пользователю по умолчанию назначается роль **sales**. Роль админа задаётся вручную в БД или через отдельный метод (например, только существующим admin).
